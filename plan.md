# 프로젝트 실행 계획 (Phase 4: 게임 확장성 부여 및 UI 확장)

## 1단계: 맵 시스템 확장
- [x] 사용자에게 보이는 맵과, 보이지 않은 상태로 엔티티 행동 추가

- [x] 맵에 무작위 엔티티 생성 시

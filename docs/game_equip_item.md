# 장비형 아이템 사전 (Equipment Item Dictionary)

본 문서는 OneBit Adventure에 등장하는 **장비형 아이템** (무기·방어구)을 정의합니다.
장비 시 즉시 스탯이 반영되며, 같은 슬롯에 중복 장착 시 기존 장비를 자동 해제합니다.
코드에서 새 아이템을 추가할 때는 `item_db.c`의 `g_item_table`에 행을 추가합니다.

---

## 장비 규칙

- 같은 슬롯에 이미 장비가 있으면 기존 장비의 스탯 보너스를 먼저 제거한 뒤 새 장비를 적용.
- DEF는 몬스터 공격 시 `damage = monster.atk - player.def` (최소 0)로 피해 감소.
- 장비를 해제하면 해당 보너스가 즉시 원상복구.

---

## 새 장비 아이템 추가 방법

1. `src/logic/item_db.c`의 `g_item_table`에 행 추가 (`atk_bonus` 또는 `def_bonus` 설정).
2. `include/item_db.h`의 `ITEM_DB_COUNT`를 1 증가.
3. 본 문서에 항목 추가.

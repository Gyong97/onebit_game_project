# 몬스터 사전 (Monster Dictionary)

본 문서는 OneBit Adventure에 등장하는 모든 몬스터를 정의합니다.
코드에서 새 몬스터를 추가할 때는 `monster_type_t` 열거형과 `monster_db.c` 테이블에도 함께 등록합니다.

---

---

## 공통 규칙

- **스케일링**: 탐험 깊이(Depth/10)에 따라 HP·ATK가 10%씩 복리 증가.
  `실제 스탯 = base * 1.1^(scroll_count / 10)`
- **스폰**: 맵 스크롤 시 상단 버퍼 행에 확률(`MONSTER_SPAWN_PCT = 30%`)로 무작위 생성.
- **사망**: HP ≤ 0이면 즉시 소멸하고 플레이어에게 XP 부여.

---

## 새 몬스터 추가 방법

1. `include/monster.h`에 `MONSTER_TYPE_NEWTYPE` enum 값과 `NEWTYPE_INIT_HP`, `NEWTYPE_INIT_ATK` 상수 추가.
2. `src/logic/monster_db.c`의 `g_monster_table`에 1행 추가.
3. 본 문서에 항목 추가.
